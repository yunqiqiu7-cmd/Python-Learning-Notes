
# 牛客网 Python 列表 NP17~NP31 知识点笔记
## NP17 生成列表
**知识点**
- 使用 `input().split()` 将空格分隔的字符串转为列表
- `split()` 默认按空白符（空格、Tab）分割
- 指定分隔符：`split(',')`、`split('\n')`

```python
lst = input().split()
print(lst)
```

**易错点**
- `split()` 无法拆分无分隔符字符串（如 `'123'` 不会变成 `['1','2','3']`）

---

## NP18 生成数字列表
**知识点**
- 先分割字符串，再用列表推导式批量转整数

```python
nums = [int(s) for s in input().split()]
print(nums)
```

**易错点**
- `input().split()` 得到的是**字符串列表**，直接运算会出错
- 不能写 `int(input().split())`，会直接报错
- 可以写 `print(input().split()))`
---

## NP19 列表长度
**知识点**
- `len(列表)` 获取元素个数

```python
lst = input().split()
print(len(lst))
```

**易错点**
- `len` 是函数，不是方法，不能写 `lst.len()`

## NP20 增加派对名单（尾部添加）
**知识点**
- `list.append(item)` 在列表末尾添加元素
- **原地操作，返回 None**

```python
lst = input().split()
lst.append("Alice")
print(lst)
```

**易错点**
- 不能在 print 里直接写 `print(lst.append(x))`，会输出 `None`
- `append/pop/insert/remove/reverse` 均为原地操作

---

## NP21 增加派对名单（头部添加）
**知识点**
- `list.insert(index, item)` 在指定索引插入元素

```python
lst = input().split()
lst.insert(0, "Bob")
print(lst)
```

**易错点**
- 同样原地操作，不可直接放 print 内

## NP22 删除第一个元素
**知识点**
- `list.pop(index)` 删除指定索引元素
- `pop()` 默认删除最后一个（`pop(-1)`）
- 删除第一个：`pop(0)`

```python
lst = input().split()
lst.pop(0)
print(lst)
```

---

## NP23 删除好友（列表筛选）
**知识点**
- 使用 `in / not in` 判断元素是否存在
- 列表推导式快速筛选

```python
a = input().split()
b = input().split()
res = [x for x in a if x not in b]
print(res)
```

---

## NP24 删除列表后三位
**知识点**
- 连续删除末尾三位，需分开三次调用 `pop()`

```python
lst = input().split()
lst.pop()
lst.pop()
lst.pop()
print(lst)
```

**易错点**
- 不能链式调用：`lst.pop().pop().pop()` 会报错

---

## NP25 有序列表（排序）
**知识点**
- `sorted(列表, key=None, reverse=False)`
- `reverse=False` 升序，`True` 降序
- **sorted 返回新列表，不修改原列表**

```python
lst = ['p','y','t','h','o','n']
print(sorted(lst))
```

**易错点**
- 与 `list.sort()` 区分：`sort` 原地排序，`sorted` 生成新列表

---

## NP26 列表反转
**知识点**
- `list.reverse()` 原地反转列表

```python
lst = input().split()
lst.reverse()
print(lst)
```

## NP27 嵌套列表
**知识点**
- 用 append 将列表作为元素添加，形成二维列表

```python
a = [1,2]
b = [3,4]
c = [5,6]
res = []
res.append(a)
res.append(b)

```

## NP28 密码拼接（无间隙拼接）
**知识点**
- 字符串可直接遍历每个字符
- `''.join(列表)` 无缝拼接
- 数字列表拼接需 `map(str, 列表)`

```python
s = input().strip().zfill(4)
# 逐位加密
nums = [(int(c) + 3) % 10 for c in s]
# 直接按交换后的顺序拼接
res = ''.join(map(str, nums))
print(res)
```

**易错点**
- 数字不能直接 join，必须先转字符串

## NP31 切片输出

**知识点**
- 切片格式：`列表[起始:结束]`
- **左闭右开**：包含起始位置，不包含结束位置
- 省略起始：从列表开头开始
- 省略结束：到列表末尾结束
- 负数索引：`-1` 最后一个元素，`-2` 倒数第二个

```python
lst = [10, 20, 30, 40, 50, 60]

# 前 n 个元素
print(lst[:n])    # 例 lst[:3] → [10,20,30]

# 最后 n 个元素
print(lst[-n:])   # 例 lst[-2:] → [50,60]

# 第 i 个到第 j 个元素（从 1 开始数）
print(lst[i-1:j])

# 全部元素（复制列表）
print(lst[:])
```

**易错点**
- 切片是**左闭右开**，结束位置不包含
- 切片不会修改原列表，返回新子列表
- 超出范围不会报错，会自动截断

# 赋值 / 引用 / 深浅拷贝 / map / list 精简补充笔记

## 1. 直接赋值（误区：误以为是拷贝）
错误理解写法
```python
lst = [1,2,[3,4]]
a = lst
b = a
```
- 无任何新对象产生
- a、b、lst 完全指向**同一个列表**

本质结论
单纯 `变量 = 变量`：
**只是引用别名，不属于拷贝，连浅拷贝都不算**

举一反三
- 执行 `append / pop / 改元素`，所有关联变量同步变化
- 重新赋值 `lst = [新内容]`，只改变当前变量，不影响其他

---

## 2. 浅拷贝 三种正规写法
冗余绕路写法
手动遍历、逐个添加元素造新列表

精简统一写法
```python
# 三种等价 全为浅拷贝
new1 = lst.copy()
new2 = lst[:]
new3 = list(lst)
```

举一反三
- 只复制**外层容器**，生成新列表
- 嵌套子列表、字典仍共用同一引用
- 修改**单层元素**互不影响；修改**嵌套内层元素**，会污染原数据

---

## 3. 深拷贝 唯一写法
手动多层遍历嵌套复刻（极繁琐）

精简写法
```python
import copy
new_lst = copy.deepcopy(lst)
```

举一反三
- 完全递归复制所有层级
- 内外全部新建，彻底隔离
- 修改任何层级元素，都不会污染原列表
- 嵌套列表/二维矩阵必须用深拷贝

---

## 4. map 函数 核心误区
不完整危险写法
```python
res = map(int, input().split())
```
res 是迭代器，**不是列表**
无法索引、无法len、无法正常打印、不能增删元素

精简完整写法
```python
# 多变量解包
a, b = map(int, input().split())

# 生成完整列表
lst = list(map(int, input().split()))
```

举一反三
- `map` 只做批量转换，不生成列表
- 需要**整体列表** → 外层必须套 `list()`
- 需要**直接拆分成单个变量** → 直接解包，不用 list()

---

## 5. list() 内置函数 实用场景
零散低效写法
循环遍历、逐个追加转换

精简用法
```python
# 1. 可迭代对象转列表
lst = list(字符串 / map / range / 元组)

# 2. 列表浅拷贝
new = list(old_lst)
```

举一反三
- `list()` 两个核心作用：类型转换 + 浅拷贝
- 纯单层数据，`list()` 拷贝安全够用
- 含嵌套可变对象，不能用，必须deepcopy
