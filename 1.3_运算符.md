# 运算符模块·精简规律总结（只抓你不精简的地方）

## 1. 输入：字符串列表 → 转 int/float
**你原来的思路**
split 得到字符串列表 → 再循环/逐个转 int
```python
parts = input().split()
a = int(parts[0])
b = int(parts[1])
```

**精简一步到位**
```python
a, b = map(int, input().split())
```

**举一反三**
- 一行完成：分割 + 类型转换 + 多变量赋值
- float 同理：`map(float, ...)`
- 多少个变量就写多少个，自动对应拆分后的列表元素

---

## 2. 多变量直接赋值：左边变量 = 右边列表
**你原来的思路**
取索引赋值，很啰嗦
```python
lst = input().split()
x = lst[0]
y = lst[1]
```

**精简写法**
```python
x, y = input().split()
```

**举一反三**
- 右边是可迭代对象（列表、split 结果）
- 左边几个变量，右边就对应几个元素
- 不用索引，不用中间列表变量

---

## 3. 多行输出简化：少写 print，用 sep
**你原来的思路**
多次 print 换行
```python
print(a + b)
print(a - b)
```

**精简写法**
```python
print(a + b, a - b, sep='\n')
```

**举一反三**
- 多个结果要换行：统一 `sep='\n'`
- 要空格：`sep=' '`（默认可不写）
- 能一个 print 解决，绝不写多个

---

## 4. 字符串输入：两行输入直接赋值
**你原来的思路**
多余变量、多余步骤
```python
s1 = input()
s2 = input()
```
其实已经很简，但你想更极致：**直接在判断里用**

**极致精简**
```python
print(input() == input())
```

**举一反三**
- 简单比较/判断，可以直接把 input 写进表达式
- 不需要先存变量

---

## 5. 成员判断：input 直接写在 print 里
**你原来的思路**
存列表 → 存要查找的值 → 判断
```python
lst = input().split()
name = input()
print(name in lst)
```

**精简一行**
```python
print(input() in input().split())
```

**举一反三**
- 简单成员判断：一行 `input() in input().split()`
- 不需要任何中间变量

---

# 最终浓缩成 4 条万能精简铁律
1. **输入+拆分+转类型**：一律 `a,b = map(int, input().split())`
2. **列表→多变量**：直接 `x,y = 列表`，不用索引
3. **多行输出**：一个 print + `sep='\n'`
4. **简单判断/比较**：input 直接写表达式里，不存变量

这就是你所有“不够精简”的地方的统一总结，
以后任何题都套这 4 条就行，不用再按题型分类。
