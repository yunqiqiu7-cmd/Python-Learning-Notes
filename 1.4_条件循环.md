## 1. 布尔自动判断规则（来源：NP41、NP43）
### 你的原始思考&踩坑
一开始想直接写 `x=bool(input())`，结果逻辑完全错误；
误以为要手动转布尔，不知道 if 后面会**自动转布尔**。

### 知识点原理
if 后面不需要手动写 `bool()`，Python 自动判断真假：
- 数字：`0`假，非0真
- 字符串：空串假，有内容真
- 容器：空列表假，非空真

### 错误写法（你踩坑）
```python
x = bool(input())
```
原因：字符串转布尔永远为真，完全错误。

### 正确写法
```python
x = int(input())
```

### 总结
不要对 input 直接转 bool；先转数字，再交给 if 自动判断。

## 3. 不区分大小写判断（来源：NP45）
### 你的原始思考&踩坑
- 不知道要不要转大小写
- 纠结能不能原地修改原列表
- 纠结要不要新建列表
- 担心修改后输出不对

### 知识点原理
1. 原地修改原列表：**不可以**，会丢失原始大小写，输出错误
2. 必须新建副本（列表推导式），不改动原数据
3. 新旧都转小写，统一对比

### 正确代码
```python
current = ["xxx"]
current_low = [x.lower() for x in current]
```

### 你的总结（保留）
不允许修改原数据，新建副本用于判断，原列表用于输出。

## 5. range 本质、迭代、列表转换（来源：NP50、NP51、NP53、NP54）
### 你的原始理解（完全正确）
1. range **不是列表**，是可迭代对象
2. range 不会一次性存所有数，循环时逐个生成
3. 必须配合 for 循环才能取出
4. 你的写法：空列表+循环+append
5. 更简洁：直接 `list(range())`

### 知识点总结
- 原始写法：
```python
lst = []
for i in range(10):
    lst.append(i)
```
- 最简写法：
```python
lst = list(range(10))
```

## 7. break & continue 原理（来源：NP59、NP60）
### 你的理解（100%正确，完全保留）
统一规则：**先判断，后打印**
- break：遇到目标停止循环，不再输出
- continue：遇到目标跳过，不输出，继续后面

### 正确结构
```python
# break
for i in ...:
    if i==目标:
        break
    print(i)

# continue
for i in ...:
    if i==目标:
        continue
    print(i)
```

### 知识点

2. 必须对每个元素单独乘
3. 可以用二维列表推导代替两层循环
```python

```
